Overview

This example features an executable language defined with xMOF.

It illustrates xMOF and the GEMOC animation framework on a Finite State Machine (FSM) language.

Getting started

Install the projects of this language:
File > new > Examples... > xMOF FSM Language

Create and start an eclipse runtime:
Run > Run Configurations... > Eclipse application > new

Install sample models for the language:
File > new > Examples... > Model Example for xMOF FSM Language
